# esm-vacunacion-app

App para el seguimiento y gestión de vacunación.

Terminología de dominio: visita = consulta, encounter = atención, appointment = cita.

## Marco normativo
- Ley N.° 28010, Ley General de Vacunas (Perú).

## Límites funcionales
- Administra la experiencia de inmunización del paciente.
- Muestra planes, módulos de vacunación y componentes asociados a campañas o seguimiento.
- No cubre la atención clínica general fuera del dominio de inmunización.
- No sustituye el registro maestro del paciente ni otros módulos preventivos especializados.

## Integraciones
- APIs y recursos de inmunización.
- Componentes de plan, dashboard y flujo de vacunación.
- Configuración y tipos compartidos del módulo.
